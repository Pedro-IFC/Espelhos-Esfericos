Aumento:

    Para caracterizar uma imagem como maior ou menor, sem representação geométrica, utilizamos o conceito de AUMENTO de uma imagem.

    Definimos o aumento de uma imagem a partir da comparação entre as alturas da imagem e objeto, ou das distâncias da imagem e objeto.

Se A>1 então a imagem é maior que o objeto

Se A<1 então a imagem é menor que o objeto

Se A=1 então a imagem é do mesmo tamanho que o objeto
